# Lakshmi Printers - Design Style Guide

## Brand Identity

### Brand Philosophy
Lakshmi Printers represents premium quality printing services with a modern, professional approach. The brand embodies reliability, craftsmanship, and innovation in the printing industry while maintaining accessibility and excellent customer service.

### Visual Language
- **Premium & Professional**: Clean, sophisticated layouts with high-quality imagery
- **Modern & Trustworthy**: Contemporary design elements that inspire confidence
- **Accessible & Friendly**: Warm colors and welcoming interface design
- **Efficient & Reliable**: Streamlined user experience reflecting quality service

## Color Palette

### Primary Colors
- **Rich Maroon**: #7A1F2F - Primary brand color, header backgrounds, primary buttons
- **Gold**: #F0C419 - Accent color, call-to-action buttons, highlights, icons

### Neutral Colors
- **Ivory White**: #F6F6F6 - Main background color, card backgrounds
- **Pure White**: #FFFFFF - Content backgrounds, text on dark backgrounds
- **Carbon Black**: #1A1A1A - Primary text color, headings
- **Cool Grey**: #707070 - Secondary text, subheadings, descriptions
- **Soft Grey**: #DADADA - Borders, dividers, subtle elements

### Optional Accent
- **Soft Cyan**: #33A1C9 - Used sparingly for special highlights or data visualization

## Typography

### Primary Font: Poppins
- **Usage**: Headings, navigation, buttons, call-to-action text
- **Weights**: 
  - Light (300) - Subtle headings
  - Regular (400) - Body text in headings
  - Medium (500) - Navigation items
  - SemiBold (600) - Main headings
  - Bold (700) - Hero headings, important CTAs

### Secondary Font: Inter
- **Usage**: Body text, descriptions, form labels, footer content
- **Weights**:
  - Regular (400) - Standard body text
  - Medium (500) - Emphasized text
  - SemiBold (600) - Strong emphasis

### Font Hierarchy
- **H1 (Hero)**: Poppins Bold, 48px/56px, #FFFFFF on dark backgrounds
- **H2 (Section)**: Poppins SemiBold, 36px/44px, #1A1A1A
- **H3 (Subsection)**: Poppins Medium, 24px/32px, #1A1A1A
- **H4 (Card Titles)**: Poppins SemiBold, 20px/28px, #7A1F2F
- **Body Text**: Inter Regular, 16px/24px, #707070
- **Small Text**: Inter Regular, 14px/20px, #707070
- **Button Text**: Poppins Medium, 16px/24px

## Visual Effects & Styling

### Layout Principles
- **Grid System**: 12-column responsive grid with 24px gutters
- **Container Max Width**: 1200px for optimal readability
- **Section Padding**: 80px vertical, 24px horizontal on mobile
- **Card Spacing**: 24px gaps between cards

### Shadows & Elevation
- **Card Shadow**: 0 4px 12px rgba(122, 31, 47, 0.1)
- **Button Shadow**: 0 2px 8px rgba(122, 31, 47, 0.15)
- **Hover Shadow**: 0 8px 24px rgba(122, 31, 47, 0.2)

### Border Radius
- **Cards**: 8px
- **Buttons**: 6px
- **Images**: 4px
- **Form Inputs**: 4px

### Interactive States
- **Button Hover**: Darken color by 10%, lift with shadow
- **Card Hover**: Scale 1.02, increase shadow
- **Image Hover**: Maroon overlay with zoom icon
- **Link Hover**: Gold underline animation

## Component Styling

### Header
- **Background**: #7A1F2F
- **Height**: 80px
- **Logo**: Left-aligned, 60px height
- **Navigation**: Right-aligned, white text, Poppins Medium
- **Call Button**: Gold background, maroon text

### Hero Section
- **Background**: Linear gradient from #7A1F2F to #5A1824
- **Text**: White text, centered
- **Buttons**: Primary (maroon), Secondary (gold)
- **Min Height**: 600px

### Service Cards
- **Background**: #FFFFFF
- **Border**: 1px solid #DADADA
- **Icon**: Gold (#F0C419), 48px
- **Title**: Poppins SemiBold, #7A1F2F
- **Description**: Inter Regular, #707070

### Gallery Grid
- **Columns**: 3 on desktop, 2 on tablet, 1 on mobile
- **Gap**: 24px
- **Hover**: Maroon overlay with opacity 0.8
- **Border Radius**: 8px

### Form Elements
- **Input Background**: #FFFFFF
- **Border**: 1px solid #DADADA
- **Focus Border**: #7A1F2F
- **Placeholder**: #707070

### Footer
- **Background**: #7A1F2F
- **Text Color**: #FFFFFF
- **Link Color**: #F0C419 (hover: #FFFFFF)
- **Social Icons**: Gold with white hover

## Animation & Motion

### Scroll Animations
- **Fade In**: Opacity 0 to 1, translateY 20px to 0
- **Duration**: 300ms ease-out
- **Trigger**: When element enters top 50% of viewport
- **Stagger**: 100ms delay between multiple elements

### Button Interactions
- **Hover**: 200ms ease-in-out
- **Active**: Scale 0.98, 100ms ease-in
- **Focus**: Outline 2px solid #F0C419

### Image Hover Effects
- **Overlay**: Fade in 200ms ease-out
- **Scale**: 1.05 transform 300ms ease-out
- **Icon**: Slide up from bottom 250ms ease-out

## Responsive Breakpoints

- **Mobile**: 320px - 767px
- **Tablet**: 768px - 1023px
- **Desktop**: 1024px - 1439px
- **Large Desktop**: 1440px+

## Accessibility Standards

- **Contrast Ratio**: Minimum 4.5:1 for all text
- **Focus Indicators**: 2px solid gold outline
- **Alt Text**: Descriptive text for all images
- **ARIA Labels**: Proper labeling for interactive elements
- **Keyboard Navigation**: Full keyboard accessibility

## Brand Applications

### Logo Usage
- **Primary**: Maroon logo on light backgrounds
- **Reverse**: White logo on maroon backgrounds
- **Minimum Size**: 120px width for digital
- **Clear Space**: 2x logo height on all sides

### Photography Style
- **Professional**: High-quality, well-lit images
- **Consistent**: Warm color temperature matching brand
- **Relevant**: Printing equipment, finished products, team
- **Composition**: Clean, uncluttered, professional

### Iconography
- **Style**: Minimalist line icons
- **Weight**: 2px stroke
- **Color**: Gold (#F0C419) or maroon (#7A1F2F)
- **Size**: 24px standard, 48px for feature icons
